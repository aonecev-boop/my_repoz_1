> ИИ-консультант мебельной студии «Встрой-Мебель.ру»

---

## 1. Концепция дизайна

### 1.1. Философия

Дизайн приложения должен передавать ценности мебельной студии:
- **Качество и надёжность** — чистые линии, стабильная геометрия
- **Современность** — актуальные UI-паттерны, плавные анимации
- **Уют и тепло** — мягкая палитра, скруглённые элементы
- **Профессионализм** — минимализм, отсутствие визуального шума

### 1.2. Стиль

| Параметр | Решение |
|----------|---------|
| Визуальный стиль | Soft UI / Glassmorphism с элементами Flat Design |
| Настроение | Элегантно, спокойно, доверительно |
| Плотность | Просторная компоновка, достаточно «воздуха» |
| Иконки | Линейные (outline), скруглённые, единый стиль |
| Углы | Скруглённые (border-radius: 12–16px для карточек, 20–24px для виджета) |
| Тени | Мягкие, многослойные (box-shadow с несколькими уровнями) |

## 5. Анимации и переходы

### 5.1. Каталог анимаций

| Элемент | Анимация | Длительность | Easing |
|---------|----------|-------------|--------|
| Открытие виджета | Scale + Fade снизу | 0.3s | ease-out |
| Закрытие виджета | Scale + Fade вниз | 0.2s | ease-in |
| Новое сообщение бота | Slide-in слева + Fade | 0.25s | ease-out |
| Новое сообщение клиента | Slide-in справа + Fade | 0.2s | ease-out |
| Кнопка отправки (клик) | Scale 0.9 → 1 | 0.15s | ease |
| Кнопка микрофона (запись) | Пульсация красного кольца | 1.5s loop | ease-in-out |
| Hover на кнопки | Scale 1.05 + тень | 0.15s | ease |
| Индикатор «печатает» | Волна из трёх точек | 1.2s loop | ease-in-out |
| Загрузка файла | Прогресс-бар + спиннер | — | linear |
| Появление бейджа | Scale 0 → 1 с отскоком | 0.3s | ease-out bounce |

### 5.2. Принципы анимации

- Все анимации — **CSS-only** (без JS-анимаций для производительности)
- Используется `transform` и `opacity` (GPU-ускорение)
- `prefers-reduced-motion: reduce` — отключение анимаций для пользователей с настройкой

## 9. Микро-взаимодействия

| Контекст | Действие | Обратная связь |
|----------|----------|----------------|
| Отправка сообщения | Клик «➤» или Enter | Сообщение появляется справа + поле очищается |
| Запись голоса | Зажатие 🎤 | Пульсирующее красное кольцо, таймер записи |
| Завершение записи | Отпускание 🎤 | Анимация «обработка» → текст появляется в поле |
| Загрузка файла | Клик 📎 → выбор файла | Прогресс-бар в области ввода → превью файла |
| Ответ бота | Получение ответа | Индикатор «печатает» → сообщение появляется (slide-in) |
| Hover на кнопку чата | Наведение мыши | Подсказка «Написать консультанту» + увеличение |
| Сворачивание чата | Клик «✕» | Плавное уменьшение → иконка |
| Новое сообщение (свёрнуто) | Бот написал ответ | Бейдж на иконке + звуковой сигнал |
| Ошибка сети | Потеря соединения | Жёлтая полоса «Нет связи. Переподключение...» |

## 12. Доступность (Accessibility)

| Требование | Реализация |
|------------|------------|
| WCAG 2.1 уровень AA | Минимальный контраст текста к фону ≥ 4.5:1 |
| Навигация с клавиатуры | Tab-фокус на всех интерактивных элементах |
| Screen Reader | Все иконки имеют `aria-label`, сообщения — `role="log"` |
| Reduced motion | `@media (prefers-reduced-motion: reduce)` — отключение анимаций |
| Focus visible | Видимый outline для клавиатурного фокуса (кольцо `#D4A84B`) |
| Размер touch-target | Минимум 44×44px для мобильных кнопок |
