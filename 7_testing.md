# 7. Тестирование

## Стратегия тестирования

### Уровни тестирования
1. **Unit-тесты** — тестирование отдельных функций и компонентов
2. **Интеграционные тесты** — тестирование взаимодействия модулей
3. **E2E тесты** — тестирование пользовательских сценариев
4. **Ручное тестирование** — проверка UI/UX

## Инструменты

| Тип | Инструмент |
|-----|-----------|
| Unit-тесты | Jest / Vitest |
| E2E тесты | Playwright / Cypress |
| API тесты | Postman / Thunder Client |
| Нагрузочные | k6 / Artillery |

## Запуск тестов

```bash
# Все тесты
npm run test

# Unit-тесты
npm run test:unit

# E2E тесты
npm run test:e2e

# Покрытие кода
npm run test:coverage
```
